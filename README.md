# Telegram VirusTotal Bot

A telegram bot which can scan telegram files smaller than 32MB.


# Instructions

  1. Hit the Deploy Button.

  2. Add your `telegram app_id`, `api_hash` and `bot_token`.

  3. Add your `virustotal_api key`.
